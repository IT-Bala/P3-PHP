Controller:

Basic controller

Controller class name should be file name and also must extends class WLT_Controller. If you want access parent methods you shoud call parent construct.

defaultController.php

<?php defaultController extends WLT_Controller { } ?>

Model:

Basic model

Model class name should be file name and also must extends class WLT_Model.


dataModel.php

<?php dataModel extends WLT_Model { } ?>

View:

Basic view

The view file should be in .phtml extension. so that set to controller without path & extension see below.


header.phtml

<html xmlns="http://www.w3.org/1999/xhtml"> <head> <=$title?> </head> <body> </body> </html>
